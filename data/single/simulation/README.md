Density profiles and chemical potential data are obtained from GCMC simulation[^1] with errors obtained from block averaging. Pressure data, structure factors[^2], and the binodal[^3] were obtained from MD simulations using LAMMPS.


 [^1]: Obtained using https://github.com/annatbui/gcmc ; see the paper for detailed methods.
 [^2]: Using the method described in [A. T. Bui and S. J. Cox, *J. Chem. Phys.*, 2024, **161**, 104103.
](https://doi.org/10.1063/5.0223750)
[^3]: See slab-geometry simulations in [K. Mazarakos, S. Qin and H.-X. Zhou, in Phase-Separated Biomolecular Condensates: Methods and Protocols, eds H.-X. Zhou, J.-H. Spille and P. R. Banerjee, Springer US, New York, NY, 2023, pp. 1–35](https://doi.org/10.1007/978-1-0716-2663-4_1).
